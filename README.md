# 🪙 Crypto Monitor

[![CI](https://github.com/clawto/crypto-monitor/actions/workflows/ci.yml/badge.svg)](https://github.com/clawto/crypto-monitor/actions/workflows/ci.yml)
[![Skill](https://img.shields.io/badge/OpenClaw-Skill-blue)](https://openclaw.ai)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

Real-time cryptocurrency monitoring skill for OpenClaw agents. Track prices, whales, trending tokens, and airdrops.

## Quick Install

```
clawhub install clawto/crypto-monitor
```

## Features

- 📊 **Real-time prices** — BTC, ETH, top 100 via CoinGecko
- 🐋 **Whale tracking** — Large on-chain transfers and exchange movements
- 🔥 **Trending discovery** — Find hot tokens before they pump
- 🪂 **Airdrop radar** — Never miss free money
- 📱 **Alert support** — Integration with messaging platforms

## Usage Examples

```
# Check prices
> What's the BTC price?

# Track whales
> Any whale movements today?

# Find trending tokens
> What's trending right now?

# Check airdrops
> Any active airdrops?
```
